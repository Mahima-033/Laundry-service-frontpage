# Laundry-service-frontpage

This project is a responsive front page for a laundry service website built using **HTML and CSS**. It focuses on clean layout design, proper structure, and modern styling techniques.

## Features
- Semantic HTML structure
- Flexbox-based navigation bar
- Full viewport-height hero section
- Clean and organized CSS
- Responsive design using media queries

## Technologies Used
- HTML5
- CSS3 (Flexbox, calc(), media queries)
